Back-Testing Lab

BUILD v4 · Robert Parker data · 208 region-years · 16 regions · includes §4 Factor Regression

Score completed past seasons from real Open-Meteo archive weather, compare against known vintage quality, and tune the model weights to fit. Because the index is relative, the headline metric is rank correlation — does the model order each region's good and bad years correctly?

About the ground-truth data: these vintage scores are from a single critic (Robert Parker Wine Advocate) used as a private internal validation key — wider range than an aggregate, so the model has real signal to fit. Don't republish these scores in your product. Weather data by Open-Meteo.com.

4 · Discover factors from data (regression)

    
Instead of hand-set weights, fit a regularized regression of Parker scores onto weather features — standardized within each region, then pooled, so it learns the real sign and weight of each factor. Cross-validated (5-fold), so the numbers below are out-of-sample skill, not memorization. Run the back-test first (it fetches & caches the weather).
